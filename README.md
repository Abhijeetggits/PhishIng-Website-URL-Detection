# Phishing-Website-URL-Detection
A smart system that detects and prevents phishing websites by analyzing suspicious URLs.

Overview / Problem:
    

 Phishing attacks are increasing rapidly, and many users cannot identify fake or malicious websites. This project helps to automatically detect whether a given URL is safe or     phishing.


 How it Works:
    

- The user inputs a URL
- The system extracts features from the URL
- A trained machine learning model analyzes the features
- The model predicts whether the URL is phishing or legitimate

Features:
- Detects phishing URLs
- Fast and accurate prediction
- Uses machine learning algorithms
- Easy to use


Tech Stack:
- Python
- Pandas
- NumPy
- Scikit-learn

How to Run:
 1. Clone the repository:
   git clone https://github.com/your-username/phishing-url-detection.git

 2. Go to project folder:
   cd phishing-url-detection

 3. Install dependencies:
   pip install -r requirements.txt

 4. Run the project:
   python app.py



 Results:
